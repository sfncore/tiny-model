# SmolLM2-135M Witness Model — Experiment Report

## Goal
Fine-tune a sub-500M parameter LLM to replace the witness agent's patrol
decision-making. The model must output valid JSON tool calls and select the
correct action for each patrol scenario.

## Environment
- Hardware: 8-core AMD EPYC, 30GB RAM, **no GPU** (CPU-only training + inference)
- Model: HuggingFaceTB/SmolLM2-135M-Instruct (134.5M parameters)
- Stack: PyTorch 2.10.0+cpu, Transformers 5.2.0, TRL 0.29.0, PEFT 0.18.1

## Output Format
Pure JSON tool calls (Format B):
```json
{"tool": "gt_nudge", "args": {"target": "gastown/furiosa", "message": "..."}}
```

Three formats tested (A: inline XML tags, B: pure JSON, C: control tokens).
Only Format B produces reliably parseable output at 135M params.

## Production Candidate: Experiment 9

**Scenario results (8/8 correct):**

| Scenario | Expected | Model Output | Latency |
|----------|----------|-------------|---------|
| idle_patrol | patrol_report/none | `none` | 1,214ms |
| healthy_polecat | peek/status/patrol | `gt_peek` | 1,511ms |
| stuck_polecat | nudge/peek | `gt_nudge` | 1,297ms |
| completed_polecat | nuke/check_git | `gt_polecat_nuke` | 1,508ms |
| crash_loop | escalate/nuke | `gt_mail_send` (to mayor) | 2,547ms |
| unpushed_work | escalate/check_git | `gt_mail_send` (recovery) | 2,692ms |
| check_infrastructure | check_tmux/status | `check_tmux_session` | 1,099ms |
| mail_check | mail_inbox/read | `gt_mail_inbox` | 829ms |

**Eval set (50 held-out real conversations, 49 assistant turns):**
- JSON valid: 100%
- Valid tool name: 100%
- Correct schema: 100%
- Latency p50: 1,561ms | p95: 3,124ms

**Training config:** full fine-tune, lr=2e-4, batch=1, grad_accum=4, max_length=2048

## Key Findings

1. **135M params learns JSON tool-call format perfectly** — 100% valid JSON
2. **Synthetic scenario data is critical** — 300 hand-crafted examples > 500 multi-turn
3. **Hardened = synthetic + real data** gives best generalization (100% eval metrics)
4. **Healthy vs stuck distinction** requires positive health signals in training data
5. **CPU inference ~1-2.5s** — acceptable for 60-90s patrol cycle
6. **Format B (pure JSON) is the only viable format** at this model size

## Dataset Composition (Experiment 9)

- **500 decision-focused examples**: extracted from real witness sessions via
  `extract_decisions.py` — (context → action) pairs ending in tool calls
- **300 synthetic examples**: generated by `synthetic_scenarios.py` — hand-crafted
  scenario templates with randomized rigs, polecats, beads, branches
